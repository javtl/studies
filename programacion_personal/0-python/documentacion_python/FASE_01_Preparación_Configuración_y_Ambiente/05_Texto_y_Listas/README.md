## FASE 1: Paso 4 - Texto y Listas Introductorias

Este paso cubre las secciones **3.1.2. Texto**, **3.1.3. Listas**, y **3.2. Primeros pasos hacia la programación**, que forman parte de la **3. Una introducción informal a Python**.

### Puntos Clave a Estudiar (Capítulos 3.1.2 y 3.1.3)

#### 1. Texto (3.1.2)

Python soporta la manipulación de cadenas de texto. El sitio web de Python ilustra ejemplos de salida simple usando `print` y cómo funciona la asignación de entrada de datos (`input`) y salida formateada.

#### 2. Listas (3.1.3)

Las listas son uno de los **tipos de datos compuestos** que Python comprende. Son fundamentales para organizar colecciones de datos.

*   Las listas son conocidas como *arrays* en otros lenguajes.
*   Las listas pueden ser **indexadas**, **segmentadas** (*sliced*) y **manipuladas** con otras funciones *built-in* (incorporadas).
*   Esta sección del tutorial las introduce informalmente como parte de "Usando Python como una calculadora".

#### 3. Primeros Pasos hacia la Programación (3.2)

Una vez introducidos los tipos de datos básicos (Números, Texto, Listas), la documentación pasa a **3.2. Primeros pasos hacia la programación**, donde se comienza a aplicar esta sintaxis elemental.

### Ejercicio Práctico

Utilizando el intérprete interactivo (que aprendiste a invocar en el Paso 2), realiza los siguientes ejercicios:

1.  **Manipulación de Texto Simple:** Crea una variable que contenga tu nombre y otra que contenga tu apellido. Únelas para mostrar tu nombre completo en una sola línea.
2.  **Lista Simple e Indexación:**
    *   Crea una lista llamada `asignaturas` que contenga los nombres de tres asignaturas de tu clase.
    *   Accede e imprime únicamente el segundo elemento de la lista (recuerda que la indexación comienza en cero).
    *   Añade una cuarta asignatura a la lista usando un método de manipulación incorporado (*built-in*) de las listas.
