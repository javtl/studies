# Mes 1 – Fundamentos sólidos de Java (4 semanas)

---

## Semana 1: Variables, tipos de datos y operadores

* **Día 1:**

  * Leer sobre tipos de datos en Java (int, double, char, boolean, String).
  * Escribir un programa simple que declare variables de cada tipo y las imprima.

* **Día 2:**

  * Practicar operadores aritméticos (+, -, \*, /, %).
  * Programa: Calculadora básica que haga suma, resta, multiplicación y división.

* **Día 3:**

  * Operadores relacionales y lógicos (`>`, `<`, `==`, `&&`, `||`).
  * Programa: Comprobar si un número es mayor que otro y si está en un rango.

* **Día 4:**

  * Práctica: Escribir un programa que pida dos números y muestre cuál es mayor o si son iguales.

* **Día 5:**

  * Repaso y ejercicios cortos:

    * Convertir temperaturas entre Celsius y Fahrenheit.
    * Calcular área de un círculo dado el radio.

* **Fin de semana (opcional):**

  * Revisar videos cortos o apuntes para reforzar.

---

## Semana 2: Estructuras de control – `if`, `switch`, bucles

* **Día 6:**

  * Leer y practicar con `if`, `else if`, `else`.
  * Programa: Clasificar edad en grupos (niño, adolescente, adulto).

* **Día 7:**

  * Practicar `switch` con un menú simple.
  * Programa: Menú para elegir una fruta y mostrar su precio.

* **Día 8:**

  * Bucles `for`: contar del 1 al 10, sumar números.
  * Programa: Suma de los primeros N números.

* **Día 9:**

  * Bucles `while` y `do-while`.
  * Programa: Pedir números hasta que el usuario introduzca 0.

* **Día 10:**

  * Ejercicios combinados:

    * Mostrar números pares entre 1 y 50.
    * Contar cuántos números negativos hay en una lista.

* **Fin de semana (opcional):**

  * Mini-reto: Juego simple de adivinar un número con pistas.

---

## Semana 3: Arrays y colecciones básicas

* **Día 11:**

  * Introducción a arrays: declarar, inicializar, recorrer.
  * Programa: Leer 5 números y mostrar el mayor.

* **Día 12:**

  * Arrays multidimensionales básicos.
  * Programa: Matriz 3x3 con números y suma de filas.

* **Día 13:**

  * `ArrayList`: añadir, eliminar, recorrer.
  * Programa: Lista dinámica de nombres con opciones para añadir y mostrar.

* **Día 14:**

  * `HashMap` básico: claves y valores.
  * Programa: Diccionario de palabras y sus definiciones.

* **Día 15:**

  * Ejercicios combinados con arrays y colecciones.

* **Fin de semana (opcional):**

  * Revisión y pequeños retos.

---

## Semana 4: Métodos y recursividad básica

* **Día 16:**

  * Crear métodos simples con parámetros y retorno.
  * Programa: Método que calcula el factorial de un número.

* **Día 17:**

  * Métodos void y sobrecarga básica.
  * Programa: Mostrar tablas de multiplicar con métodos.

* **Día 18:**

  * Introducción a la recursividad.
  * Programa: Fibonacci con recursividad.

* **Día 19:**

  * Práctica con recursividad: suma de elementos de un array.

* **Día 20:**

  * Mini proyecto: Gestor de tareas simple por consola con métodos para añadir, mostrar y eliminar tareas (sin POO todavía).

* **Fin de semana (opcional):**

  * Repaso general y preparar dudas para preguntar.
